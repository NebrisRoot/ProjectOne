Треную Git

Команди:
git init - створення репозиторію у тій дерикторії у котрій знаходимось.

git status - виводить перелік файлів з позначенням, які мають зміни, чи взагалі не завантажені.

git add file.js - додає файл file.js

git add . - додає усі файли у директорії.

git commit -m "Completed Chapter 1" - команда комітить опис до файлів котрі завантажені\оновлені.

git log - виводить дані з описом закоміченого.

git diff Chapter1.txt - надає дані про відмінності у файлі з їх переліком.

git checkout Chapter1.txt - завантаження останньої завантаженної версії.

git remote add origin https://github.com/NebrisRoot/ProjectOne.git  - вигрузка даних у онлайн репозиторій.
git push -u origin main

git rm --cached -r . - видалить файли з репозиторію

Створення гілок розробки.

git branch (Назва)- створення нової гілки розробки.

git branch - перелік гылок розробки.

git checkout (Назва) - пеключитись на гілку з назвою (Назва)
